# UrbanSound8K_Ses_Verisi_Uzerinde-_Siniflandirma
Ses Verileri üzerinde zaman-genlik grafiği,farklı sınıflardan seslerin zaman-frekans görselleri,Mel-Spektrogram Özniteliklerinin İncelenmesi yapılmıştır.
Bu işlemlerden sonra makine öğrenmesi Random Forest yöntemi ile eğitilerek doğruluk ve kesinlik değerleri bulunarak karşılaştırma yapılmışıtr.
